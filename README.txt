README

When opening this folder please make sure that the following files are present:  
-Reddit_scrape.py
-BERT.py
-emoji_scrape.py
-Streamlit_visuals.py
-Combine_csv_Outputs.ipynb
-Visuals.ipynb
-Fine_tune.ipynb
-Experiments.ipynb
These files provide the code used for this project and create the csv files that help execute the programs along the way.

Python Libraries used:
-pandas
-numpy
-matplotlib
-sklearn
-praw
-transformers
-streamlit
-tensorflow
 
This project is based around using a BERT models from huggingface (goemotions and hateful speech detector) 
to preform sentiment analysis to try and detect hateful speech and fan emotions using comments pulled from Reddit. 
The data is displayed on a local host website throught streamlitapp.
